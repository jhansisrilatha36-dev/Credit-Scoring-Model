🚀 CreditWise AI – Creditworthiness Prediction System
CreditWise AI is a Machine Learning-based 💡 credit scoring system that predicts an individual's creditworthiness using financial data such as 💰 income, 📉 debt ratio, 💳 credit utilization, and 📊 payment history.
The project leverages an optimized 🌲 Random Forest Classifier to assess credit risk and support data-driven lending decisions. It covers the complete ML pipeline including preprocessing, feature engineering, model training, evaluation, and ⚡ real-time prediction via an interactive dashboard.
✨ Features
🔍 Creditworthiness prediction
🧹 Data cleaning & preprocessing
🧠 Feature engineering
🌲 Optimized Random Forest model
📊 Feature importance analysis
📉 Confusion Matrix & ROC Curve visualization
